# RPG Turn-Based Game — План реалізації

## Концепція гри
Покрокова (turn-based) RPG-гра, де гравець керує командою з 4 героїв різних класів.
Бої складаються з 3 хвиль ворогів, де остання хвиля містить Еліту або Боса.

---

## Класи героїв

### 1. Нападник (Attacker)
- **Роль:** головне джерело шкоди
- **HP:** низьке (80)
- **Атака:** висока (20–30)
- **Здібності:**
  - `Strike` — сильний удар по одному ворогу
  - `Poison Slash` — атака + дебафф "Отрута" (шкода кожен хід, 3 ходи)
  - `Armor Break` — атака + дебафф "Зламана броня" (ворог отримує +50% шкоди, 2 ходи)
- **Cooldown:** Strike — 0, Poison Slash — 2, Armor Break — 3

### 2. Підтримка (Support)
- **Роль:** баффи союзників, зняття дебаффів
- **HP:** середнє (100)
- **Атака:** низька (8–12)
- **Здібності:**
  - `Quick Hit` — слабка атака по одному ворогу
  - `Inspire` — бафф "Натхнення" одному союзнику (+30% атаки, 3 ходи)
  - `Purify` — знімає всі дебаффи з одного союзника
  - `Shield Buff` — бафф "Щит" одному союзнику (-30% вхідної шкоди, 2 ходи)
- **Cooldown:** Quick Hit — 0, Inspire — 2, Purify — 3, Shield Buff — 3

### 3. Лікар (Healer)
- **Роль:** відновлення здоров'я команди
- **HP:** середнє (90)
- **Атака:** низька (6–10)
- **Здібності:**
  - `Tap` — слабка атака по одному ворогу
  - `Heal` — лікує одного союзника (відновлює 25 HP)
  - `Group Heal` — лікує ВСІХ союзників (відновлює 15 HP кожному)
  - `Revive` — воскрешає полеглого союзника з 30% HP
- **Cooldown:** Tap — 0, Heal — 1, Group Heal — 4, Revive — 5

### 4. Танк (Tank)
- **Роль:** захист команди, провокація ворогів
- **HP:** високе (160)
- **Атака:** низька (10–15)
- **Здібності:**
  - `Bash` — атака по одному ворогу
  - `Taunt` — провокація: всі вороги атакують Танка наступні 2 ходи
  - `Fortify` — бафф собі: -50% вхідної шкоди на 1 хід
- **Cooldown:** Bash — 0, Taunt — 3, Fortify — 2

## Система бою

### Порядок ходів
- Кожен хід гравець обирає дію для ОДНОГО героя (по черзі для кожного живого героя)
- Після ходів всіх героїв — ходять вороги
- Порядок: Нападник → Підтримка → Лікар → Танк → Вороги

### Хвилі бою
Кожен бій складається з **3 хвиль**:

| Хвиля | Склад ворогів | Особливість |
|--------|--------------|-------------|
| 1 | 2–3 звичайних ворога | Розігрів |
| 2 | 3–4 звичайних ворога | Складніше |
| 3 | 2 звичайних + 1 Еліта/Бос | Фінальна |

### Між хвилями
- **Зберігається:** поточне HP героїв, cooldown здібностей
- **Скидується:** всі баффи і дебаффи (і на героях, і на ворогах)

### Еліта vs Бос
| | Еліта | Бос |
|--|-------|-----|
| HP | x2 від звичайного | x4 від звичайного |
| Атака | x1.5 від звичайного | x2.5 від звичайного |
| Коли з'являється | Легка/середня складність | Важка складність або кінець кампанії |

## Баффи та дебаффи

### Баффи (на союзників)
| Назва | Ефект | Тривалість |
|-------|-------|------------|
| Натхнення (Inspire) | +30% атаки | 3 ходи |
| Щит (Shield) | -30% вхідної шкоди | 2 ходи |
| Фортифікація (Fortify) | -50% вхідної шкоди (тільки Танк) | 1 хід |

### Дебаффи (на ворогів)
| Назва | Ефект | Тривалість |
|-------|-------|------------|
| Отрута (Poison) | 5 шкоди на початку ходу ворога | 3 ходи |
| Зламана броня (Armor Break) | +50% вхідної шкоди | 2 ходи |
| Провокація (Taunt) | ворог атакує тільки Танка | 2 ходи |

## Вороги

### Звичайні вороги
| Тип | HP | Атака | Особливість |
|-----|-----|-------|-------------|
| Гоблін | 40 | 8–12 | Швидкий, атакує випадкового |
| Скелет | 50 | 10–14 | Середній |
| Орк | 70 | 12–18 | Сильний, повільний |

### Еліти та Боси
- Еліти та Боси — це підсилені версії звичайних ворогів
- Боси мають спеціальну здібність (наприклад, АоЕ-атака по всій команді)

|--------|-------------------|---------------|-----------------|
| Легка | x0.8 | x0.8 | Еліта |
| Середня | x1.0 | x1.0 | Еліта |
| Важка | x1.3 | x1.3 | Бос |

## Технічний стек
- **Frontend:** HTML + CSS + JavaScript (vanilla)
- **Фреймворк:** без фреймворку — чистий JS для простоти
- **Рендеринг:** DOM-маніпуляції (не Canvas)
- **Збереження:** localStorage для прогресу
